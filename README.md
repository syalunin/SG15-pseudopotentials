This repository contains the SG15 Optimised Norm-Conserving Vanderbilt (ONCV) pseudopotentials generated by a modified D. R. Hamann scalar relativistic code. The code has been modified so that the pseudopotentials now provide the wavefunctions of the core states for all atoms. Except that these pseudopotentials are identical to those obtained by M. Schlipf and F. Gigi, now known as the SG15 collection.

The core wavefunctions are included in the pseudopotential file as a separate section PP_GIPAW. The section also contains information about the orbital momentum and the principal quantum number of the orbitals, so that Quantum ESPRESSO can read them and use them in calculations. Theoretically, these wavefunctions can be used to calculate NMR shielding tensors, EFG tensors and other magnetic resonance properties by DFT methods, e.g. as it is implemented in the QE-GIPAW package. However, I have not tested all pseudopotentials for this purpose, so use them at your own risk without any guarantee from my side.

The core states are contained in GIPAW
PP_GIPAW gipaw_data_format="2">
 <PP_GIPAW_CORE_ORBITALS number_of_core_orbitals="1">
   <PP_GIPAW_CORE_ORBITAL.1
       type="real"
       size="602"
       columns="4"
       index="1"
       label="1S"
       n="1" l="0">
