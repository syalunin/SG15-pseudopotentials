
# SG15 ONCV pseudopotentials with core electrons

This repository contains the SG15 Optimised Norm-Conserving Vanderbilt (ONCV) pseudopotentials, generated using a modified version of the scalar-relativistic code by D. R. Hamann. I modified [his code](http://www.quantum-simulation.org/potentials/sg15_oncv/) to ensure that the pseudopotentials also include the wavefunctions of core electrons for all atoms. Except for this, these pseudopotentials are identical to those generated by M. Schlipf and F. Gigi (SG15 collection).

The core wavefunctions are written in a separate section of the pseudopotential file in GIPAW format, enabling Quantum ESPRESSO to read and use them in calculations. This feature has been tested. The core wavefunctions can also potentially be used to calculate NMR shielding tensors, EFG tensors, and other magnetic resonance properties using the [QE-GIPAW package](https://github.com/dceresoli/qe-gipaw). Full compatibility with this package, however, has not yet been tested.
